# UMT-pythonweb-hw-04

Домашня робота №4: асинхронне сортування файлів за розширеннями.

## Опис

Скрипт рекурсивно читає всі файли з вихідної папки та копіює їх у директорію призначення. У директорії призначення автоматично створюються підпапки за розширеннями файлів.

Наприклад:

```text
photo.jpg  -> output/jpg/photo.jpg
text.txt   -> output/txt/text.txt
archive.gz -> output/gz/archive.gz
README     -> output/no_extension/README
```

Файли не видаляються з вихідної папки, а саме копіюються.

## Запуск

```bash
python sort_files.py <source_folder> <output_folder>
```

Приклад:

```bash
python sort_files.py ./source ./output
```

## Особливості

- використовується `argparse` для аргументів командного рядка;
- використовується `asyncio` та `asyncio.gather`;
- копіювання файлів виконується через `asyncio.to_thread`;
- підтримується рекурсивний обхід підпапок;
- файли без розширення потрапляють у папку `no_extension`;
- помилки логуються через `logging`;
- при збігу назв файлів створюється унікальна назва, щоб не перезаписати файл.
